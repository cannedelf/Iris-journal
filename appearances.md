# Sunnyside CAS Reference 🌈
*Complete reference for Create-A-Sim!! Appearances, aspirations, personality points and genetics all in one place!!*
*Updated as each Sim is planned or created!!*

---

## How To Use This File
1. Create PARENTS first for any Sim with hidden genetics!!
2. Generate Sim from parents in CAS!!
3. Adjust appearance to match notes!!
4. Set aspiration!!
5. Set personality points!!
6. ✅ Mark as done!!

---

## Key Rules
- Always create parents in CAS first for Sims who need hidden recessive genes!!
- Iris has NO hidden genes — she is purely and completely herself!!
- Total personality points = 25!!
- See sims2.md genetics section for full hidden gene planning!!
