## How To Use This File
1. Create PARENTS first for any Sim with hidden genetics!!
2. Generate Sim from parents in CAS!!
3. Adjust appearance to match notes!!
4. Set aspiration!!
5. Set personality points!!
6. ✅ Mark as done!!
